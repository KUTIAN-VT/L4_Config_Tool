# L4 配置工具运行包

本仓库用于发布 L4 配置工具的 Windows 运行包。运行包内已经包含主程序、后台服务、固件升级工具、Qt 运行库、Web 界面资源和使用手册，完整解压后即可运行。

适用对象：

- 现场调试、生产测试或售后维护人员。
- 需要查看 L4 设备状态、配置链路参数、执行对频或固件升级的用户。

完整操作说明请查看：[手册](docs/L4配置工具软件使用手册.md)。

## 获取软件

普通用户请从 [L4 配置工具 Releases 页面](https://github.com/KUTIAN-VT/L4_Config_Tool/releases) 下载完整运行包：

1. 打开 Releases 页面并选择需要的版本。
2. 在 `Assets` 区域下载 `L4_Config_Tool-vX.Y.Z.zip` 格式的压缩包。
3. 完整解压 ZIP 文件。
4. 进入解压后的目录，运行 `L4_Config_Tool_Web.exe`。

注意：

- 不要使用仓库首页的 `Code` -> `Download ZIP`。该方式下载的是源码，不是完整运行包。
- 不要下载 Release 中自动生成的 `Source code (zip)` 或 `Source code (tar.gz)`。这些文件通常不包含可执行文件和运行库。
- 不要单独复制或下载 `L4_Config_Tool_Web.exe`。软件运行依赖同目录下的 DLL、`web/`、`resources/`、`platforms/`、`l4_daemon.exe` 等文件。

## 运行步骤

1. 确认运行包已经完整解压，目录结构没有被拆分。
2. 将 L4 设备连接到电脑，并保持稳定供电。
3. 双击 `L4_Config_Tool_Web.exe` 启动软件。
4. 等待底部状态栏显示 `l4_daemon 已连接`。
5. 在右上角设备列表中选择目标设备，再进行信息查看、链路配置、固件升级或配置维护。

## 目录说明

| 路径 | 说明 |
| --- | --- |
| `L4_Config_Tool_Web.exe` | L4 配置工具主程序 |

| `web/` | 配置工具 Web 界面资源 |

## 常用功能

- 查看设备信息、固件版本、MAC 地址和运行状态。
- 管理对频信息和对端设备。
- 查看链路指标和环境扫频结果。
- 配置频点、频宽、发射功率、MCS 等链路参数。
- 执行本机或远程固件升级。
- 导入、导出和维护配置文件。

## 常见问题

### 双击后无法启动

- 确认下载的是 Release 中的完整运行包，而不是源码 ZIP。
- 确认没有单独移动 `L4_Config_Tool_Web.exe`。
- 安装运行包内的 `vc_redist.x64.exe` 后重新启动软件。

### 一直显示 daemon 未连接

- 确认 `l4_daemon.exe` 与主程序在同一目录。
- 关闭软件后重新启动 `L4_Config_Tool_Web.exe`。
- 检查是否有安全软件拦截后台服务启动。

### 设备列表为空

- 检查设备供电、USB/串口连接和驱动状态。
- 更换连接线或 USB 端口后重新打开软件。
- 确认设备处于可维护状态。

## 操作注意事项

- 固件升级、写配置和对频操作会改变设备状态，请在设备处于可维护状态时执行。
- 升级或写入配置过程中不要断电、拔线或关闭工具。
- 现场出现异常时，先截图保存底部 `运行日志` 和设备状态，再进行重启或重新插拔。
- 如果需要保存链路变化过程，请在 `链路监控` 页面点击 `记录`，结束后点击 `导出` 保存 CSV 文件。
